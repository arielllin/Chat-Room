# Chat-Room

A live demo can be found on [this link](https://arielllin.github.io/Chat-Room/#/).

## Tech Stack

* Vue.js
* Firebase Storage

## How to Use

```bash
# Clone this repository
$ git clone https://github.com/arielllin/Chat-Room.git

# Go into the repository
$ cd Chat-Room

# Install dependencies
$ npm install

# Run the app
$ npm run dev
```
